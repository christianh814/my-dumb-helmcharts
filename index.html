Welcome to my Helm charts repository!

Usage

Add this Helm repository to your Helm client:

helm repo add my-charts https://christianh814.github.io/my-dumb-helmcharts/
helm repo update

Search for Charts

helm search repo my-charts

Install a Chart

helm install my-release my-charts/<chart-name>

Available Charts

Check the index.yaml file for all available charts.
